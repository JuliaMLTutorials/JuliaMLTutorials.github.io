Machine Learning With Julia

Are you ready to discover the joys of machine learning with Julia? Perhaps you’ve heard about Julia’s first-class performance? Perhaps you read an article about Julia being the future of machine learning? Maybe you’re just sick of Python? Well this blog is for you!

This collection of tutorials are aimed at those with previous experience with machine learning, who are interested in exploring Julia’s capabilities in the field. Each tutorial takes the form of a Pluto notebook which can be run in the cloud with Binder or downloaded to be run on your own machine. A link to each dataset will be provided in the tutorial description.
